📌 Project Description

  • Part of a recruitment platform used by hiring managers and recruiters
  • Centralized interface for viewing, tracking, and managing candidates
  • Displays name, resume, and interview details
  • Includes search and filter functionality

📈 Business Requirements

  • Intuitive interface for viewing and managing all candidates
  • Search by name, email, or phone
  • Display key info: name, resume, interview details
  • Enable quick access to resumes and interview links
  • Ensure PII data accuracy and security

⚙️ Functional Requirements

Candidate Listing Table
  • 10 candidates per page with pagination
  • Columns: Name, Resume, Interview Details
Search Bar
  • Located at the top of the page
  • Supports: name, email, phone number
Data Handling
  • Dynamically loads based on search or pagination
  • Resume column shows PDF icon if uploaded
  • Interview column links to detailed view
Responsiveness
  • Page layout must be responsive across devices
